bower 软件下载器使用流程 1、安装 npm install bower@1.8.2 -g 2、查看帮助 bower --help 3、创建bower的配置文件 .bowerrc 4、初始化 初始化完成后 会生成bower.json的配置文件 bower init 5、通过bower 来安装angularjs bower install angular@1.3.8 --save 6、安装项目所需的插件库 bower install angular-ui-router@0.2.11 --save 路由 bower install ng-file-upload@12.2.13 --save 文件上传 bower install ng-file-upload-shim@12.2.13 --save 进度条 bower install angular-cookies@1.2.9 --save 存放cookie
